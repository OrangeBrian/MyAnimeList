# Project Setup
Clone repository. </br>
Create database. </br>
Import the script SQL, use <b>schema-mariadb.sql</b> on <i>"resources/template"</i>. </br>
Don't forget to configure the application properties file to avoid errors. </br>
<b>mariadb / XAMPP</b> is required for this project.
